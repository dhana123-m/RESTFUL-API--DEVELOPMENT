# 📚 Result API - Library System

A RESTful API to manage a book inventory using Node.js, Express, and a JSON file as database.

---

## 📦 Tech Stack
- Node.js
- Express.js
- JSON (file-based database)

---

## ▶️ Getting Started

1. Clone or download the repo
2. Run in terminal:
   ```bash
   npm install
   node server.js
